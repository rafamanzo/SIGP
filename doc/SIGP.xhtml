SIGP


    Bem vindo ao site do SIGP, um projeto criado por alunos do IME-USP como parte disciplina MAC332. SIGP é um sistema de informações para grupos de pesquisa que facilita o gerenciamento de grupos, membros e publicações acadêmicas. O SIGP é software livre.

    Este projeto é baseado nas tecnologias Java, Hibernate, VRaptor, Tomcat, e MySQL. Os desenvolvedores usam o Eclipse como ferramenta para desenvolvimento.


DESENVOLVIMENTO

    O código fonte do SIGP é disponibilizado no github. Para começar, baixe o projeto e importe-o no Eclipse. Você precisará de um servidor Tomcat e banco de dados MySQL para rodar o sistema.

https://github.com/rafamanzo/SIGP


    O sistema Pivotal é utilizado para gerenciar o desenvolvimento, e pode ser acessado no endereço:

https://www.pivotaltracker.com/projects/354269
